# healthsoftware
Health Diagnosis using the Ruffier Method

A simple application that evaluates cardiovascular fitness using the Ruffier Index.
The program helps users estimate their physical condition by analyzing heart rate measurements taken before and after exercise.

Overview

The Ruffier Method is a well-known fitness assessment technique used to evaluate how the heart responds to physical effort.

The application:

Collects heart rate values from the user
Calculates the Ruffier Index
Interprets the result
Displays a health/fitness diagnosis

This project can be used for:

Educational purposes
Sports and fitness monitoring
Health awareness projects
Programming practice
How the Ruffier Test Works

The test is based on three pulse measurements:

Measurement	Description
P1	Resting heart rate before exercise
P2	Heart rate immediately after exercise
P3	Heart rate after recovery

The Ruffier Index formula:

IR=
10
(P1+P2+P3)−200

Result Interpretation
Ruffier Index	Physical Condition
< 0	Excellent
0 – 5	Very Good
5 – 10	Good
10 – 15	Average
> 15	Poor
